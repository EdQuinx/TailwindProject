# TailwindCSS CoffeeStyle Project by EdQuinx

## Requirements

| Tech          | Version    |
| ------------- | ---------- |
| **[Node.JS]** | >= v14.7.0 |
| **[npm]**     | >= v6.14.7 |
| **[yarn]**    | >= v1.19.1 |

### Guide for this repository is here:

Clone this repository and run test

```sh
git clone https://github.com/EdQuinx/TailwindProject.git <project_dir>
cd <project_dir>
yarn build:css
yarn start:dev
```
